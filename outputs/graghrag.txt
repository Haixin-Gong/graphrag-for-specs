# 电力系统接地技术规格书
## 1. 总则
本技术规格书依据中国国家标准GB/T 50065《交流电气装置的接地设计规范》及相关行业标准制定，适用于各类电气装置的接地系统设计、施工及验收工作。规范旨在确保接地装置的安全性和可靠性，防止人身触电事故和设备损坏[Data: Reports (21); Sources (19, 21)]。
## 2. 引用标准
- GB/T 50065-201X《交流电气装置的接地设计规范》
- GB 50169-2006《电气装置安装工程 接地装置施工及验收规范》
- DL/T 380《接地降阻材料技术条件》
- IECTR 61400-24《风力涡轮发电机系统 雷电防护》
- IEC 62305-1:2010《雷电防护系统要求》[Data: Entities (193, 209, 399); Relationships (189, 194, 195)]
## 3. 接地装置基本要求
### 3.1 接地极选择
- 自然接地极优先采用建筑物基础钢筋、埋地金属管道等
- 人工接地极应采用热镀锌钢或铜覆钢材料，锌层厚度≥0.5mm，铜层厚度≥0.254mm
- 严禁使用金属软管、管道保温层金属外皮等易腐蚀材料[Data: Sources (21); Entities (436); Relationships (189)]
### 3.2 接地电阻要求
- 一般电气装置接地电阻值应符合GB/T 50065规定
- 风力发电机组防雷接地电阻应≤10Ω（依据IECTR 61400-24）
- 光伏发电站组件金属边框可作为接闪器，需满足厚度要求（铝≥0.65mm，钢≥0.5mm）[Data: Reports (153); Sources (24); Relationships (195, 196)]
## 4. 特殊场所接地要求
### 4.1 风力发电系统
- 接地系统需包含系统接地、保护接地和雷电保护接地
- 接地装置应满足DL/T 380对降阻材料的技术要求
- 多台风电机组应建立互联接地系统[Data: Entities (211); Relationships (193, 194, 195)]
### 4.2 爆炸性环境
- 必须采用符合GB50058-2014标准的防爆电气设备
- 接地系统设计应考虑防闪电感应措施
- 接地装置应与电气和电子系统共用接地装置[Data: Reports (21); Entities (368); Relationships (360, 981)]
## 5. 施工与验收
### 5.1 施工要求
- 接地极埋深≥0.5m，冻土区应考虑季节变化影响
- 接地线连接必须采用可靠焊接或专用连接器
- 需设置测量井分离自然与人工接地极[Data: Sources (19, 21); Entities (193)]
### 5.2 验收标准
- 接地电阻测试应在装置施工后尽快进行
- 需检查接地线的机械强度和防腐性能
- 风力发电机组和光伏电站需进行专项验收[Data: Sources (19); Relationships (158, 159, 160+more)]
## 6. 附录
### 6.1 接地电阻计算
环形接地体工频接地电阻计算公式：
R = 2ρ/3d (Ω)，其中d=√(A/π)，ρ为土壤电阻率(Ω·m)，A为环形接地体包围面积(m²)[Data: Sources (48); Entities (556)]
### 6.2 相关标准版本说明
GB/T 50065与《交流电气装置的接地设计规范》为同一标准不同版本[Data: Relationships (190)]
注：本规格书未涵盖高压直流输电系统接地要求，该类系统应参照专用规范执行[Data: Sources (19); Relationships (183)]。
